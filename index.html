The Multi-Module Security System

The Multi-Module Security System was a prototype project designed to replace an existing security
system with a less expensive alternative that is capable of doing multiple different tasks including
motion detection and door status detection. The intent was to create a system capable of remotely
notifying the house owner of any impending situations.

The project was created by a team of 3 as a project requirment for University of Chicago CS 342
for the semester of Spring 2017. It was initially proposed by Joseph as it had been a personal idea
of his for some time before. After settling on Joseph's initial idea, The team of 3 ran with the idea
both presenting it and bringing the MMSS idea through a compressed Alpha testing and Beta
Testing cycle.
